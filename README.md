# LiftCode 🛗

Herramienta de diagnóstico rápido para códigos de error de ascensores 
y escaleras mecánicas Schindler y MONARCH.

## Características
- Búsqueda por código, título o descripción
- Filtros por categoría y severidad
- Favoritos persistentes
- Historial de últimos 8 códigos consultados
- Modo oscuro / claro

## Stack
- Kotlin + Jetpack Compose
- Material Design 3
- DataStore Preferences
- MVVM + Repository Pattern

## Marcas soportadas
- Schindler
- MONARCH

## Plataforma
Android — minSdk 24
